Monet 1-Page Site Design and front-end implementation (HTML/CSS/JS)

Timeframe
June 2023, 1 week
Team
Solo
What I did
UI, Front‐end (HTML/CSS/JS)
Tools
Figma, Visual Studio Code, Brackets, W3 HTML and CSS validator

Project Goal

Create a 1-page site with 3 required features ...with "creation" entailing both the design and the front-end implementation.

Content focus: I decided to create a webpage for the painter Claude Monet. This webpage would display a selection of his paintings, briefly introduce the painter, and sell Monet memorabilia.

Required features: Light/dark mode; form validation; product display and shopping cart with cost calculation.

Design Goals

Create a design that has a unifying visual theme, is usable, and is aesthetic.

Design Concept

Elegant and sleek, like a museum listing, with tamer colors that won't distract from Monet's vibrant artworks.

Colors: Black, white, and grays

Typefaces: Metal, Lora, Rubik

I went with an elegant look in part because I wanted to try something new and get away from my comfort zone of sans-serif plus bright colors.

UI Design

After settling on the overall visual style, I developed the final design by creating a wireframe and then high-fidelity mockups.

Design Highlights

Web users tend to scan, not read. The design serves this habit via:

  • Visual hiearchy like headings, and visually showing thematic content grouping.
  • Clear visual affordances (interactive elements look interactive, etc).
  • Drawing out and emphasizing user-desired information (such as the number of items in the cart or the total dollar amount).

Features Showcase

All the required features were sucessfully implemented according to specifications

Light/Dark Mode

This features allows users to switch between the color schemes

Form Validation

User input is checked to ensure it is in the right format and that all required information is provided. Note the swapping requirement between phone or email depending on the user's prior selection.

Shopping: Cart, Calculator, and Checkout

Visual representation of adding or removing products to an online shopping cart with the total changing to match. Checkout is represented with an alert to the user.

Reflection

Learning from past mistakes, I made sure to plan out the CSS beforehand, which made the coding process much smoother and prevented major roadblocks and struggles.

If I'd had more time, I would want add more fun effects like scroll or hover effects. I'd also want to add a lightbox feature for the paintings for a better viewing experience.
