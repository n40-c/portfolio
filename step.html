Step Product Design Responsive e-commerce site design

Timeframe
Oct 2023, 1 week
Team
Solo
Fields
UI, UX thinking
Tools
Figma

Project Goal

Create a responsive design for e‑commerce webpage, following client’s branding and needs.

The client is Step, a (made-up) shoe seller.

Scope: product list, login/signup, menu
Deliverables: high fidelity mockups in 3 sizes (mobile, tablet, and pc)

Design Goals

  • Organize client-required features in an intuitive manner for user’s ease of use.
  • Enable efficiency by minimizing user clicks.

With only a week, this project would be a challenge.

Client Requirements

Client requirements included branding, styles and content.

Typefaces: Gatwick, Public Sans

Colors: a bright green (7ed957) and a dark gray (545454)

Content: 18 required ideas (ideas such as search bar, email sign-up, careers page, etc), the brand’s logo, product images, product descriptions, and so on.

Ideation + Process

I ideated on feature placement, ease of interaction, and visual hierarchy.

Grouping Features

To get started, I organized the features into groups according to where users would expect to find them.

Layout in Low-Fidelity

I ideated on layout with low-fidelity mock-ups. I decided on placement based on importance of the element, user expectations, and the available space.

My aim was for intuitive access & consistent layout across the 3 sizes.

High-Fidelity Mockups

I iterated on the final UI design and created high-fidelity mockups.

Flow Chart

I also made flow charts to show how the mockups connect for the menu.

User-Centered Highlights

I kept the user in mind throughout the design process from site/page architecture to the visual styling. Two key highlights include:

Combining icons, color, and text

Utilizing a combination of text, icon, and color in tandem to provide dependencies and ensure users can easily identify the info they need.

User-friendly inputs

Picking User-friendly filter inputs for each data type, including adding quality of life features like a search bar for certain filters.

Final Mockups

Completed mockups for responsive site. 3 sizes with ancilliaries like the menu.

Reflection

If I had more time, I would work on the visual balance of the designs. I’d also work on making the login/sign up experience more consistent across the 3 sizes.

In this project, I learned the importance of low-fidelity mock-ups, especially the time-saving benefits of layout and placement.
